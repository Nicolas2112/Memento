Les commandes utiles

Man Permet de recevoir des informations supplémentaire sur la commande.

ls permet d'afficher la liste des fichier et dossier
Autres options avec ls :
-a ou -A permet d'afficher les fichiers cachés
-l affiche la version longue
-al Couple les deux précédentes options

pwd permet d'afficher le chemin du dossier courant

cd et cd .. permet de se déplacer dans l'arborescence des dossiers

sudo -s permet de se mettre en super utilisateur.

apt update met à jour la liste des paquets

apt upgrade met à jour les paquets

apt search cherche un élément dans la liste de paquets

apt install permet d'installer un paquet spécifique

dpkg Dépackage un paquet

mkdir créé un dossiers

mkdir -p ***/*** créé un sous dossier et son sous dossier

mkdir *** *** *** créé plusieurs dossiers

touchpermet de créer un fichier

nano (avec fichier) lance l'éditeur nano

mv permet de déplacer un fichier/dossier ou de le renommer

rm supprime le document ou le fichier (dangereux !!!)

shift+insert autre façons de coller dans linux (natament nano)

x-www-browser lance le navigateur par défaut

firefox lance firefox
